# Jay Yadav - Personal Portfolio

Welcome to the repository for my personal portfolio website, live at [thejayyadav.github.io](https://thejayyadav.github.io).

## About This Project
This repository contains the static source code for my professional website. It serves as a digital resume and project showcase, highlighting my five-year background in Civil Engineering and my active transition into the IT and Data Center sector.

* **About:** A brief overview bridging my infrastructure project management experience with my passion for technology.
* **Skills Dashboard:** Highlighting technical proficiencies including Python scripting, Linux environments, PC hardware troubleshooting, and civil quantity estimation.
* **Featured Projects:** * Python-based security scripts and file decryption tools.
  * Freelance civil engineering consulting (estimation and billing).
  * Financial data analysis concepts.
* **Resume Integration:** Direct download link for my latest tailored CV.

## Technologies Used
* HTML5
* CSS3
* Hosted via GitHub Pages

## Local Viewing
Since this is a static site built without a framework, no complex installation or build processes are required to view the code locally.
1. Clone this repository to your local machine.
2. Open the `index.html` file directly in any modern web browser.
